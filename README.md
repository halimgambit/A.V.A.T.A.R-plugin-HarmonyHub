# HarmonyHub
Harmony hub vous donne le pouvoir de contrôler vos appareils en IR

This plugin is an add-on for the [A.V.A.T.A.R](https://avatar-home-automation.github.io/docs) framework.

🎯 Usage
Commandes :

allume la télé, 
éteins la télé, 
allume la play station, 
mets netflix

remplacer l'adresse ip du harmony hub dans le fichier config

# Multi-room
The HarmonyHub plugin is fully multi-room.

# Multi-language
The HarmonyHub plugin relies solely on the system's available languages.
